## 📁 Featured Project

### [TPRM: AI Customer Support Vendor Assessment](https://github.com/stokeskene/tprm-ai-vendor-assessment)

A full TPRM lifecycle applied to 10 AI customer support platforms on behalf of a simulated healthcare client. Built from scratch — including a purpose-built 52-question DDQ with AI-specific controls not found in standard templates.

**Frameworks:** NIST AI RMF 1.0 · NIST SP 800-161r1

**What's included:**
- Custom DDQ template mapped to NIST controls across 9 security domains
- Pre-filled vendor DDQs sourced from public trust centers (Conveyor, SafeBase, Vanta)
- NIST framework mapping matrix and vendor risk tracker
- Vendor outreach templates and follow-up documentation
- Active assessments: Zendesk (complete), Intercom (DDQ issued), Ada (DDQ issued)
